Вход

Эндпоинт: POST /login

Этот эндпоинт используется для входа пользователя и получения JWT-токена для аутентификации.

Параметры запроса:

  • email: Электронная почта пользователя (обязательный)
  • password: Пароль пользователя (обязательный)

Пример запроса:

{
    "email": "example@example.com",
    "password": "your_password"
}
